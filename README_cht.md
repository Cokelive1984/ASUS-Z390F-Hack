### 安裝
1. 確認BIOS版本與設定組態, 我的設定組態(./BIOS/v{版本號}_setting.txt), 也可以直接導入我的設定值透過我匯出的文件(./BIOS/v{版本號}_config.CMO)
2. 建置USB系統安裝碟, 並替換EFI分區為我發佈的文件.
